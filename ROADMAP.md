# Polyview 로드맵

> 향후 작업의 **컨텍스트를 보존**하기 위한 문서다 — 무엇을·왜·어떤 설계로 갈지.
> 작업을 나중에 다시 시작할 때 여기부터 읽으면 된다.

## 제품 비전

AI 시대에 누구든 만들어내는 다양한 산출물(Markdown·HTML·코드·다이어그램)을 **사람이
읽기 좋은 형태로 표현**하고, **협업 중 생각을 빠르게 공유**하는 도구. 만료되는 짧은
링크는 *빠른 공유의 수단*이고, 프로젝트가 끝나면 그 산출물들이 **SSOT / 코드 기반
RAG 의 근거**로 이어진다. 타깃은 개발자에 국한되지 않는다 — AI 로 산출물을 뽑는
모든 사람. 코어는 **오픈소스(셀프호스트)**, 편의는 **유료 클라우드**(open-core).

## 현재 상태 (v0.1.1)

- Tauri 2 데스크탑 앱. 로컬 디렉터리 / GitHub 저장소의 파일을 런타임에 읽어 렌더
  (Markdown·OpenAPI·TSX·HTML·이미지·코드).
- 정제된 네이티브 UI(상단바 제거, 사이드바 푸터, 브레드크럼·상태바), 로컬 파일
  우클릭(경로 복사 · Finder 에서 보기).
- GitHub Actions 릴리스 자동화(원클릭 `workflow_dispatch`), 무서명 배포.

## 향후 기능

### 1. 공유 (Sharing) — 다음 핵심 ★

로컬 산출물을 **랜덤 짧은 링크**로 공유하고 **기본 3일 후 만료**. 받는 사람은 브라우저에서
렌더된 사본을 본다(앱 없이).

- **범위**: 단일 파일.
- **접근 제어**: `link`(링크 아는 사람) / `emails`(지정 이메일) / `domain`(같은 도메인, 예 `@socar.kr`).
- **아키텍처 결정**:
  - **저장**: 원본 + 참조 에셋 업로드 → 웹 뷰어가 *데스크탑과 동일한 렌더러*로 렌더
    (정적 스냅샷 아님 → 렌더링 코드베이스 1개 재사용, 백엔드는 dumb).
  - **백엔드**: thin (Rust axum 권장) — 짧은 ID + 메타 DB(Postgres, 최소 구성 SQLite) +
    오브젝트 스토리지(클라우드 R2/S3, 셀프호스트 MinIO/디스크). 큰 에셋은 pre-signed URL 로
    클라이언트가 스토리지에 직접 업로드.
  - **만료**: 스토리지 lifecycle + DB TTL 스윕으로 N일 뒤 blob+메타 삭제(휘발성 = 프라이버시).
  - **보안 (가장 중요)**: 공유 콘텐츠는 신뢰 불가(남이 만든 HTML/TSX 실행). 반드시
    **별도 origin**(예 `usercontent.<domain>` 또는 `<id>.share.<domain>`)에서
    **sandbox iframe(allow-scripts, same-origin 금지) + 엄격 CSP** 로 격리해 앱/API 쿠키·
    다른 공유 접근 차단. PAT·로컬 경로 등 **시크릿은 절대 업로드 금지**.
  - **인증 = 교체 가능 provider** (OSS 는 계정 불필요, 남용 제어는 공개 클라우드만의 문제):
    | 모드 | 대상 | 발행자 | 열람자 | 이메일/도메인 제한 | 남용 제어 |
    |---|---|---|---|---|---|
    | `none` | OSS 개인 | 설정 토큰 | 없음(링크=권한) | ❌ 링크 전용 | 운영자 책임 |
    | `oidc` | OSS 팀 | 사내 IdP | 사내 IdP | ✅ IdP 보장 | 내부망·IdP |
    | `managed` | 클라우드 | 매직링크·OAuth | 매직링크·OAuth | ✅ | 계정별 쿼터·레이트리밋 |
  - **열람자 인증 게이트**(제한 공유): 매직링크 / OAuth `hd`(hosted domain, 예 Google
    Workspace `@socar.kr`) / 사내 SSO. provider 가 `none` 이면 `emails/domain` 옵션은 비활성.

### 2. 지식화 — SSOT / RAG

프로젝트 완료 후 **보존 전환된 공유**들을 프로젝트/컬렉션으로 묶어 **임베딩 → 벡터 DB
색인 → RAG**. 지금은 데이터 모델에서 **ephemeral(만료) vs retained(보존)** 분리만 해 두면 된다.

### 3. 배포 품질

- macOS **코드 서명·공증**(Apple Developer ID, 연 $99) → "확인되지 않은 개발자" 경고 제거.
  현재는 무서명이라 우클릭→열기 우회가 필요(README 참고).
- Windows 코드 서명(SmartScreen 경고 제거), Tauri 자동 업데이터(GitHub Releases 기반).

## 미해결 결정

- **브랜드 이름**: 현재 `Polyview` 유지. 새 브랜드로 가려면 — AI 네이밍 포화로 사전 단어·조어
  대부분 선점됨. 32개 검증 중 충돌이 가장 적었던 건 조어 **`Limnly`**("limn = 또렷이
  그려내다 → 읽히게", 발음 마찰은 약점). 이름은 보류.
- **무료 티어 익명 허용 여부**: 익명은 남용 제어가 불가해 **계정** 쪽으로 기움. OSS 는 토큰
  또는 OIDC 로 대체(위 인증 표 참고).
